Server

Before you install make sure your server meets the following requirements:

  • ReadyPOS requires PHP v8.1 + and MySQL 5.7 or MariaDB 10.0
  • Intl PHP Extension
  • Intl Curl PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • XML PHP Extension
  • JSON PHP Extension
  • Zip Extension
  • sodium Extension
  • BCMath Extension

In most servers these extensions are enabled by default, but you should check with your hosting provider.

WARNING

Please do this very carefully. We will not be responsible for your mistake.
